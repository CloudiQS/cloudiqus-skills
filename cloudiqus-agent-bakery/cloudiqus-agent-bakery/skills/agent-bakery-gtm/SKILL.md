# Agent Bakery GTM Skill

## What Agent Bakery Is
A productised service for building and deploying production AI agents on AWS Bedrock AgentCore.
Not chatbots. Not demos. Working agents that do real business processes.

## The Three Components
1. **Agent Bakery Factory** — where agents are built (AWS Bedrock AgentCore + FAST template)
2. **StoreFront / AWS Marketplace** — where agents are discovered and purchased
3. **Unify Portal** — where customers manage and monitor their agents

## ICP (Ideal Customer Profile)
- AWS customer, digital-first, 100–2,000 employees
- Has a repetitive business process costing >£5K/month in human time
- Technical enough to understand AI value, not technical enough to build it themselves
- Sector: financial services, education, manufacturing, healthcare, retail

## Pricing Model
| Tier | What | Price |
|------|------|-------|
| Discovery | Process analysis, agent design | £5–10K fixed |
| POC | Working agent in their environment | £15–30K (POC credits reduce to £5–15K net) |
| Production | Full deployment + monitoring | £30–100K |
| Per Agent | Ongoing management | £500–2,000/month per agent |

## Sales Motion
1. Lead with the cost of the problem: "What does this process cost in human hours per month?"
2. Show a working example: FinOps agent, document processing agent, onboarding agent
3. POC credits: "AWS can fund up to $25K of the POC cost"
4. Fast time-to-value: "Working agent in 4–6 weeks"

## Talking Point
"Generic AI gives you text. Agent Bakery gives you an agent that actually does the work — connected to your systems, running on AWS, monitored and maintained by CloudiQS."
