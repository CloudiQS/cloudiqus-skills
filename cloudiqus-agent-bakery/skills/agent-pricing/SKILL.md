# Agent Pricing Skill

## Pricing Structure

### Build (one-time)
| Complexity | Description | Price |
|------------|-------------|-------|
| Quick Suite | Pre-built, minimal customisation | £5–15K |
| Custom — Simple | Single process, 1–2 integrations | £15–30K |
| Custom — Complex | Multi-step, 3+ integrations | £30–100K |
| Enterprise Platform | Unify Portal + multiple agents | £100K+ |

### Deploy & Manage (monthly)
| Agent Type | Monthly Management | What's Included |
|------------|-------------------|----------------|
| Quick Suite agent | £300–2,000/month | Monitoring, updates, support |
| Custom agent | £500–3,000/month | Monitoring, updates, retraining |
| Multi-agent platform | £2,000–5,000/month | Full platform management |

### AWS Funding Offset
- POC credits: up to $25K off the build cost
- MAP: if agent replaces legacy infrastructure, may qualify
- Typical net build cost after POC credits: £0–20K (makes it very easy to say yes)

## Presenting Pricing
1. Start with the cost of the problem: "You spend £X/month on this process"
2. Show the agent cost: "The agent costs £Y/month to manage"
3. Show payback: "Payback in Z months"
4. Show funding: "AWS can fund £W of the build cost"
5. Net summary: "You invest £[net] and save £[X] — that's £[saving] in year one"

## Discount Rules
- Standard: no discount without Steve approval
- Strategic customer: up to 15% for 12-month commitment
- AWS co-sell: PDM may unlock additional co-investment
- Never discount the monthly fee — it erodes margin permanently
