# StoreFront Skill

## What StoreFront Is
The public-facing catalogue of CloudiQS agents — available via AWS Marketplace and cloudiqus.com.
Customers discover, trial, and purchase agents without a sales conversation.

## AWS Marketplace Listing
Each Quick Suite agent is listed on AWS Marketplace:
- Category: Professional Services / AI/ML
- Pricing: SaaS subscription (per-month per deployment)
- Trial: 14-day free trial where possible
- Billing: through AWS bill (frictionless for AWS customers)

## Listing Requirements per Agent
- Clear use case description (one sentence)
- Before/after: what changes when the agent runs
- Pricing tiers (starter / standard / enterprise)
- Setup time: "Live in X weeks"
- Customer testimonial or proof point
- AWS services used

## Discovery Channels
1. AWS Marketplace organic search
2. CloudiQS website agent catalogue
3. PDM referrals ("CloudiQS has an agent for that")
4. Outreach (SDR agent pitches specific agents to ICP)
5. Existing customer expansion (FinOps customer adds Security Sentinel)

## The Self-Serve Motion
Future state: customer finds agent on Marketplace → deploys to their AWS account → CloudiQS manages it → bills monthly via AWS.
No sales call needed for Quick Suite agents <£1,000/month.

## Revenue Model
- Quick Suite: £300–2,000/month per agent, recurring
- Custom build: £15–100K one-time + ongoing management fee
- Platform fee (Unify Portal): £200/month per customer (future)
