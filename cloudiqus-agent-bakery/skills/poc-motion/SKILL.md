# POC Motion Skill

## The POC Journey
Week 0: Discovery — identify the process, quantify the cost, confirm POC credits
Week 1: Design — process map, agent design, data sources, integration points
Week 2: Build — agent built in CloudiQS environment using AgentCore + FAST
Week 3: Test — customer tests the agent with real data
Week 4: Refine — feedback incorporated, edge cases handled
Week 5–6: Deploy — agent running in customer's AWS environment
Week 6: Review — outcomes documented, full engagement proposal made

## Demo Script (first meeting)
1. "What's the most repetitive process your team does that you wish an AI could handle?"
2. [Listen and map to Quick Suite or custom]
3. "Let me show you the FinOps agent we built — it does [X] automatically" [live demo]
4. "We can have something like this running in your environment in 4 weeks"
5. "AWS POC credits can fund up to $25K of the build cost — let me check your eligibility"

## POC Scope Template
```
Process: [what the agent will do]
Inputs: [data sources, APIs]
Outputs: [what the agent produces]
Success Criteria: [how we know it works]
Excluded: [what's out of scope]
AWS Services: [Bedrock, AgentCore, Lambda, etc.]
Timeline: [4-6 weeks]
Cost: £[X] gross / £[Y] net after POC credits
```

## Success Criteria Examples
- "Agent processes 100 test documents with >90% accuracy"
- "Agent produces monthly cost report in <60 seconds"
- "Agent handles 50 test customer queries without escalation"

## Post-POC Conversation
"The POC proved [X]. The full production deployment would [scope]. Investment: £[Y]. Ready to proceed?"
