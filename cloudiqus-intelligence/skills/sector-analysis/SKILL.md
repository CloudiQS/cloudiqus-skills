# Sector Analysis Skill

## Education
**AWS footprint:** S3, EC2, RDS, SageMaker for learning platforms
**Pain:** Budget constraints, compliance (GDPR, Jisc), student data protection, scaling for intake periods
**CloudiQS angle:** OCRE subcontractor status, Education competency, WAFR for research compute
**Hook:** Cost savings + Security + Agent Bakery (AI tutoring, student support agents)
**Key contacts:** IT Director, Head of Digital, CTO

## Financial Services
**AWS footprint:** Multi-account, heavy compliance, encryption everywhere
**Pain:** FCA compliance, data residency, audit trails, cyber risk
**CloudiQS angle:** Security assessment, WAFR, compliance architecture
**Hook:** Security + Migration (away from legacy infra)
**Key contacts:** CTO, CISO, Head of Technology

## Manufacturing (including Luis Prato/Klexu — Spain/EU)
**AWS footprint:** IoT, edge computing, ERP integration, supply chain
**Pain:** Digital transformation pressure, supply chain visibility, sustainability reporting
**CloudiQS angle:** Agent Bakery (supply chain agents), Migration (legacy systems), FinOps
**Hook:** Agentic AI + Cost Savings
**Key contacts:** CTO, COO, Head of Digital

## Retail / eCommerce
**AWS footprint:** High-traffic web, seasonal scaling, personalisation
**Pain:** Black Friday scaling, cart abandonment, personalisation, cost during off-peak
**CloudiQS angle:** Cost optimisation (seasonal patterns), Agent Bakery (recommendation agents)
**Hook:** Cost Savings + Agent Bakery
**Key contacts:** CTO, Head of Engineering, CDO

## Healthcare
**AWS footprint:** FHIR, HL7, patient data, imaging storage
**Pain:** CQC compliance, NHS integration, patient data security, digital transformation
**CloudiQS angle:** Security assessment, WAFR, Migration from on-prem
**Hook:** Security + Migration
**Key contacts:** CTO, CISO, Head of Digital
