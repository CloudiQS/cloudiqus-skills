---
description: View, add, or update lessons in the CloudiQS lessons system. Run at start of session to load prior corrections.
argument-hint: "[view / add: [lesson text] / search: [topic]]"
---
Load skills: `lessons-system`, `autonomy-rules`
If argument is 'view': display all current lessons.
If argument starts with 'add:': append the lesson in the correct format with today's date.
If argument starts with 'search:': find lessons relevant to that topic.
If no argument: display all lessons and remind the user to review them.
