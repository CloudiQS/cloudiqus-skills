# ACE Opportunity Lifecycle

## The 7 Stages

| Stage | Status | Key Action | Funding Trigger |
|-------|--------|-----------|----------------|
| 1. Qualified | Lead | Register ACE immediately on HOT (75+) score | Check MAP eligibility |
| 2. Discovery | Active | PDM intro call, co-sell flag if >$100K | Request POC approval if GenAI |
| 3. Proposal | Active | SOW shared with customer | Confirm MAP type=Migration |
| 4. Negotiation | Active | Commercial terms, AWS co-sell support | MDF pre-approval if applicable |
| 5. Verbal Commit | Won | Customer says yes verbally | WAFR schedule if in scope |
| 6. Contract Signed | Closed Won | SOW signed, kickoff scheduled | All funding docs prepared |
| 7. Delivery | In Progress | Update ACE with milestones | MAP credits claimed per milestone |

## ACE Registration Rules
- Register as type=**Migration** for any migration project (unlocks MAP)
- Register as type=**New Business** for net-new customers
- Register as type=**Expansion** for upsells to existing customers
- AWS account ID required — get it early
- Partner-led vs co-sell: flag co-sell if PDM engagement adds value

## PDM Engagement Triggers
Contact PDM when:
- Deal value > $50K
- Migration scope > $100K (MAP territory)
- GenAI/Agent Bakery scope (POC approval needed)
- Customer wants AWS executive engagement
- Competitive situation (Azure/GCP)
