# Partner Scorecard Skill

## Advanced Tier Requirements (current)
| Requirement | Status | Notes |
|-------------|--------|-------|
| ACE opportunities | Track monthly | Quality scored by AWS |
| AWS certifications | 18+ engineers | Monitor expiry dates |
| Competency validation | Annual | GenAI, Migration, Workloads, Education, SMB |
| Customer references | 2+ per competency | Keep updated |
| Revenue threshold | Track against target | AWS measures partner-influenced revenue |

## Competency Health Check (quarterly)
For each competency:
- Certifications required: how many do we have? Any expiring?
- Case studies: are they current? Customer still active?
- Tooling validation: any changes needed?
- Renewal date: when does it expire?

## ACE Pipeline Quality
AWS scores ACE quality based on:
- Completeness (all fields filled)
- Timeliness (registered before work starts)
- Outcome accuracy (won/lost correctly marked)
- Co-sell engagement (PDM involved on large deals)

## Quarterly Scorecard Output
1. Tier requirements: MET / AT RISK / BREACH for each
2. Competency status: GREEN / AMBER / RED
3. Cert headcount: current vs required, expiring within 90 days
4. ACE quality score (if available from Partner Central)
5. Funding received YTD: MAP, WAFR, MDF, POC
6. Actions to maintain/improve tier
