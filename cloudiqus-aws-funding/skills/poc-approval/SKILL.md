# POC Approval Skill

## What POC Credits Cover
New GenAI, ML, or technically innovative workloads where the customer hasn't done it before on AWS.
Classic CloudiQS use: Agent Bakery POC, first Bedrock deployment, new ML pipeline.

## Eligibility Criteria
- Customer is new to this type of workload on AWS
- Project scope is innovation/experimentation, not production migration
- CloudiQS is the AWS partner delivering the work
- Project has not yet started

## The Non-Negotiable
**PDM written approval BEFORE the SOW is sent.**
Verbal approval is not enough. Email confirmation is the minimum.

## Approval Process
1. Brief the PDM: project name, customer, scope, estimated AWS spend, partner value-add
2. Request written approval via email: "Please confirm POC credit approval for [project]"
3. Attach PDM approval email to SOW as appendix
4. Include credit amount in the funding section of SOW

## Approval Email Template
```
Subject: POC Credit Approval Request — [Customer Name] [Project Name]

Hi [PDM name],

Requesting POC credit approval for the following:
- Customer: [name]
- Project: [description]
- Scope: [brief technical scope]
- Estimated AWS spend: $[amount]
- CloudiQS engagement value: £[amount]
- Start date: [date]

Please confirm approval so we can proceed with the SOW.

Thanks, [name]
```

## Credit Value
Up to $25,000 or 10% of total project value — whichever is lower.
