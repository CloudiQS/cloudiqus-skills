# WAFR Credits Skill

## The 5-Day Rule
Submit to AWS Partner Portal within **5 business days** of review completion.
Late = rejected. No exceptions. No appeals.

Set calendar reminder the day the review is scheduled: "WAFR CREDIT SUBMISSION DEADLINE — [Customer]"

## Credit Values
| Scope | Credit |
|-------|--------|
| 1–2 pillars | £5,000–£10,000 |
| 3–4 pillars | £10,000–£18,000 |
| All 6 pillars | £18,000–£25,000 |

## Partner Portal Submission Checklist
- [ ] Log into AWS Partner Central
- [ ] Navigate: Well-Architected → Reviews → Submit
- [ ] Customer AWS account ID entered
- [ ] Review completion date entered
- [ ] Pillar scores entered (HIGH/MEDIUM/LOW findings count per pillar)
- [ ] Findings report attached (PDF)
- [ ] Remediation plan attached
- [ ] Customer sign-off email attached
- [ ] Submit and note reference number

## Output
1. Submission deadline (5 business days from review date)
2. Credit estimate
3. Complete checklist with status
4. Required documents list
5. Calendar reminder text
