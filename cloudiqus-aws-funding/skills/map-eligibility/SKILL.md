# MAP Eligibility Skill

## Eligibility Questions
1. Is the customer migrating workloads to AWS? (If NO — MAP doesn't apply)
2. Is project value > $25K estimated? (If NO — check with PDM)
3. Has ACE been registered as type=Migration? (If NO — register NOW)
4. Has billable work already started? (If YES — flag as RED, escalate to Steve)

## ACE Registration Checklist
- [ ] Customer name confirmed
- [ ] AWS account ID confirmed
- [ ] Migration scope documented (which workloads, from where)
- [ ] Estimated AWS spend post-migration calculated
- [ ] Project start and end dates agreed
- [ ] Opportunity type set to: **Migration**
- [ ] Partner-led vs co-sell confirmed with PDM

## Credit Estimate
Typically 25% of estimated AWS spend during migration, capped at $2M.
Example: 50 VMs → £150K/year AWS spend → 6-month migration → £75K in-scope → ~£18K credits

**Always confirm actual value with PDM.**

## Output
1. Eligibility verdict: ELIGIBLE / CHECK PDM / NOT ELIGIBLE
2. ACE registration checklist with status
3. Credit estimate (indicative)
4. Immediate next action
5. RED flag if work has started without ACE
