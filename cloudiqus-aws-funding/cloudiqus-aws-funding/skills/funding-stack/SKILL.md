# Funding Stack Skill

## Cardinal Rule
Check funding before every SOW. Funding missed after work starts is rarely recoverable.

## Programme Matrix

| Programme | Value | Eligibility | Critical Rule |
|-----------|-------|-------------|--------------|
| MAP | Up to $2M credits | Migration >$25K | ACE as Migration BEFORE work starts |
| POC | Up to $25K / 10% | New GenAI/ML workload | PDM written approval BEFORE SOW sent |
| WAFR Credits | £5K–£25K | Any WAFR review | Submit Partner Portal within 5 business days |
| MDF | 50% co-fund | Marketing with AWS | Pre-approval required, claim within 60 days |
| Activate | Up to $100K | VC-backed startups | CloudiQS registers as Provider |
| BOX | DR funded pilot | FSx ONTAP managed DR | Feasibility study first |

## Stacking Rules
✅ Can combine: MAP + WAFR · MAP + MDF · POC + WAFR
❌ Cannot combine: MAP + POC on same scope · Multiple MAP on same workload

## Output for Any Opportunity
1. Programme eligibility (YES/NO/MAYBE each)
2. Total potential funding (£/$ as appropriate)
3. Immediate actions required today
4. Deadlines and gotchas
5. Best stacking strategy
6. Net customer cost after funding
