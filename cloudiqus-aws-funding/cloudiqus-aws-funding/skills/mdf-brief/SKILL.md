# MDF Brief Skill

## Eligible Activities
- AWS co-branded events and webinars
- Content creation (case studies, whitepapers) featuring AWS
- Digital campaigns with AWS messaging
- Demand generation activities with AWS co-sell intent

## Co-Funding Rules
- Typically 50% AWS / 50% CloudiQS
- Pre-approval required before committing spend
- Claim within 60 days of activity completion
- Activity must be completed as approved — no post-hoc changes

## Brief Template
```
Activity Name: [name]
Date: [date]
Estimated Cost: £[amount] total / £[amount] AWS portion
Activity Type: [event / content / campaign / demand gen]
AWS Messaging: [how AWS is featured]
Target Audience: [sector, titles, geography]
Expected Outcomes: [leads, registrations, content downloads]
AWS PDM Approval: [name, date approved]
```

## Claim Documentation Required
- Pre-approval email from PDM
- Invoices for all spend
- Evidence of activity (screenshots, attendee list, content produced)
- Lead/outcome report

## Output
1. Complete MDF brief for the activity
2. Pre-approval email draft to PDM
3. Claim deadline (60 days from activity date)
4. Document checklist
