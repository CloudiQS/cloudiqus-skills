# BOX Feasibility Skill

## What BOX Is
AWS BOX is a managed disaster recovery offering built on Amazon FSx for NetApp ONTAP.
CloudiQS is assessing it as a productised DR service for MSP customers.
Status: feasibility study in progress — check with Steve before pitching to customers.

## Customer Fit Profile
Good BOX candidates:
- Has on-premises or co-located data that needs DR
- Currently using NetApp ONTAP storage
- Has compliance/regulatory DR requirements
- Mid-market (not too small, not too large to want managed DR)
- Already an MSP customer or warm prospect

## Feasibility Assessment Questions
1. What is the customer's current DR solution (if any)?
2. What is their RTO/RPO requirement?
3. Do they use NetApp ONTAP currently?
4. What data volumes need protecting?
5. Is DR compliance-driven (ISO, Cyber Essentials, FCA, GDPR)?
6. What is their current DR cost?

## TCO Comparison (once data available)
Compare: current DR cost (on-prem or co-lo) vs AWS BOX (CloudiQS managed)
Typical value prop: lower CapEx, better RTO, compliance-ready

## Output
1. Customer fit score (HIGH/MEDIUM/LOW)
2. Feasibility assessment answers with gaps
3. Estimated TCO comparison
4. Recommended next step
5. Flag for Steve: PROCEED / HOLD / MORE INFORMATION NEEDED
