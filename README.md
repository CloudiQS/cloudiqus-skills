# CloudiQS Skills Marketplace

**59 skills · 40 commands · 8 plugins** — the operating system for CloudiQS, AWS Advanced Consulting Partner.

## Install in Cowork (recommended, no coding)

1. Open **Cowork** desktop app
2. Click **Customize** (bottom-left)
3. Go to **Browse plugins → Personal → +**
4. Select **Add marketplace from GitHub**
5. Enter: `cloudiqus/cloudiqus-skills`
6. All 8 plugins install automatically

## Install in Claude Code (terminal)

```bash
claude plugin marketplace add cloudiqus/cloudiqus-skills
claude plugin install cloudiqus-toolkit@cloudiqus-skills
claude plugin install cloudiqus-commercial@cloudiqus-skills
# ... or install all 8
```

## Install manually (Claude.ai Projects — no GitHub needed)

1. Unzip this file
2. Go to claude.ai → Projects → New Project
3. Upload the plugin folders you want into the project knowledge
4. Start a conversation and type any command
